## Overview
Modbus Mechanic is a Modbus TCP and RTU master and slave simulator that interprets 32 bit numeric register values such as int32 and float32. It can also intrepret ASCII strings of any length.

Powered by Java, it can be run on Mac, Windows, or Linux with the OpenJDK.

## Requirements
OpenJDK
RS-485 serial adaptor (for RTU)

## Installing Java
The OpenJDK provides an open source version of the Java Runtime Environment. If you do not need an installer package, you can download the binaries from Oracle's site. The Java community maintains builds that have installers for Windows, Mac and Linux. They can be downloaded here.
